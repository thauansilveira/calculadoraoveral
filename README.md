# Calculadora de Overall - PES6

Calculadora web para calcular o overall de jogadores do PES6 baseado em diferentes posições.

## Como usar localmente

1. Baixe todos os arquivos (index.html, style.css, script.js)
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Pronto! A calculadora está funcionando

## Como hospedar online

### Opção 1: GitHub Pages (Recomendado - Grátis)

1. Crie uma conta no [GitHub](https://github.com) (se não tiver)
2. Crie um novo repositório
3. Faça upload dos arquivos (index.html, style.css, script.js)
4. Vá em Settings > Pages
5. Selecione a branch "main" e clique em Save
6. Sua calculadora estará disponível em: `https://seu-usuario.github.io/nome-do-repositorio`

### Opção 2: Netlify Drop (Mais fácil - Grátis)

1. Acesse [netlify.com/drop](https://app.netlify.com/drop)
2. Arraste e solte a pasta com os arquivos
3. Pronto! Você receberá um link instantaneamente

### Opção 3: Vercel (Grátis)

1. Acesse [vercel.com](https://vercel.com)
2. Faça login com GitHub
3. Clique em "Add New Project"
4. Importe seu repositório ou faça upload dos arquivos

### Opção 4: Compartilhar arquivos diretamente

Você pode compactar os arquivos em um ZIP e compartilhar:
- Por email
- Google Drive
- Dropbox
- Qualquer serviço de compartilhamento de arquivos
